Front-end Web Development made easy.

Ashton is a front-end web develpment project framework that scales across all devices, Ashton's boilerplate markup is fast, simple and easy to customize. With less than 1800 lines of code, Ashton is easy to maintain and expand on, making it the perfect framework to start a responsive project with.

Responsive Grid

Using FlexBox, we use container, rows, and columns.

.column

Ashton is based off of Flexbox grid system with simple to use columns for creating fast responsive webpages and is based on the 12 column system. Example here:

                    <div class="container">
    <div class="row">
        <div class="example-column">
            .column
        </div>
    </div>
</div>

Images

Similar to Bootstraps images, Ashton, uses the .img-responsive class to make images responsive across all devices.

Use the .img-thumbnail for an added border to the image for thumbnails.

Adding the .img-circle class makes an image a circle.

Adding the .img-rounded gives the image rounded edges.

Tables

# Pokemon Type
001 Bulbasaur Grass
004 Charmander Fire
007 Squirtle Water

Used Bootstrap 4's model for tables to create a beautiful table which includes an inverse option for a darker finish.

Just use the .table class.

                    <table class="table">
                        <thead>
                            <tr>
                                <th>#</th>
                                <th>Pokemon</th>
                                <th>Type</th>
                            </tr>
                        </thead>
                        <tbody>
                            <tr>
                                <th scope="row">001</th>
                                <td>Bulbasaur</td>
                                <td>Grass</td>
                            </tr>
                            <tr>
                                <th scope="row">004</th>
                                <td>Charmander</td>
                                <td>Fire</td>
                            </tr>
                            <tr>
                                <th scope="row">007</th>
                                <td>Squirtle</td>
                                <td>Water</td>
                            </tr>
                        </tbody>
                    </table>

Just use the .table-inverse class to change the color to a dark theme.
